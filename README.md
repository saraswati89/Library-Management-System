## Setup
1. Clone:
   git clone https://github.com/saraswati89/Library-Management-System.git
2. Backend:
   cd backend
   mvn spring-boot:run
3. Frontend (if present):
   cd frontend
   npm install
   npm start
4. DB:
   Create database `library_db` and import `database/schema.sql`
